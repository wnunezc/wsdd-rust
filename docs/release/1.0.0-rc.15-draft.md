## Highlights

- `RC15` advances `WP-05` with safer deploy/remove recovery behavior
- canonical SSL path unified to `C:\WSDD-Environment\Docker-Structure\ssl`
- SSL references aligned across deploy, backup/restore, mkcert helpers, README, and in-app help
- deploy/remove now snapshot and restore mutated files before rollback instead of deleting blindly
- remove now reverts WSDD-managed hosts entries and SSL files for the project domain
- remove no longer recreates a missing PHP container just to process project cleanup
- `options.yml` handling hardened to repair partial state and avoid leaving orphaned `external: true` lines
- project backup/restore now supports the canonical flat SSL layout and remains compatible with legacy `cert.pem` / `key.pem` backups
- `WP-05` closed by explicit user confirmation after local validation

## Suggested release summary

WSDD `1.0.0-rc.15` continues `WP-05` by making project deploy/remove flows safer to retry after failures. This build unifies SSL handling under Docker's real working directory, restores modified files from snapshots during rollback, cleans up project hosts/SSL state during remove, and hardens `options.yml` editing to avoid partial state drift.
